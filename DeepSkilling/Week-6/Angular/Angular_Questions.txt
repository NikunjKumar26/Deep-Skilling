How to Use This Exercise Book
This exercise book covers Angular (v20.0) for the Digital Nurture 5.0 Deep Skilling Program. The 10 
hands-on exercises progress from project setup and core Angular concepts through to forms, HTTP 
integration, state management with NgRx, and unit testing.

capability to the same application rather than starting fresh each time.
